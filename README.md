# 🎰 LuckyDrawr — Lotto Number Generator

A simple and fun web-based lotto number generator built for **entertainment and casual use**.

Built with **HTML, CSS, and JavaScript** — lightweight, fast, and easy to use.

---

## 🚀 Live Demo

* 🌐 Try it here: **[https://luckydrawr.netlify.app/](https://luckydrawr.netlify.app/)**

---

## ✨ Features

### 🎲 Number Generator

* Generates random lotto numbers instantly
* One-click number creation for quick results
* Clean and readable output display

### 🔁 Regeneration System

* Regenerate new sets of numbers anytime
* Fast and responsive updates

### 📱 User Interface

* Simple and minimal design
* Mobile-friendly layout
* Easy-to-use controls for all users

## 📝 Usage

### For Users

1. Open the live demo link
2. Click the generate button
3. Get your random lotto numbers
4. Regenerate anytime for new results

### For Developers

#### 🎨 Customize Theme

```css id="luckydrawr1"
:root {
  --primary-color: #ffcc00;
  --background-color: #0f0f1a;
  --text-color: #ffffff;
}
```

#### ⚙️ Modify Logic

* Number generation → `script.js`
* UI design → `styles.css`
* Layout → `index.html`
